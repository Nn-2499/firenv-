# 🔥 firenv- Termux Setup

![Preview](https://private-user-images.githubusercontent.com/208726278/574655807-390189f5-100e-41de-83ba-2dd25adc600f.jpg)

> สคริปติดตั้ง Termux แบบ All-in-One พร้อม NvChad + Zsh + Starship
> โทนสีเขียว-ม่วง จัดเต็ม!

## ✨ Features
- ✅ Zsh + Starship prompt สองบรรทัด
- ✅ NvChad (Neovim config สวยงาม)
- ✅ ปลั๊กอิน NvimTree, Telescope, Which-key, Indent-blankline
- ✅ MOTD สีสัน ต้อนรับสวยงาม
- ✅ Nerd Fonts + ไอคอน

## 📦 การติดตั้ง

```bash
git clone https://github.com/Nn-2499/firenv-.git
cd firenv-
bash install.sh
```

🚀 หลังติดตั้ง

1. รีสตาร์ท Termux
2. พิมพ์ nvim เพื่อเปิด Neovim (ปลั๊กอินจะติดตั้งอัตโนมัติ)
3. ใน nvim รัน :MasonInstallAll และ :TSInstallAll

🎨 Keybindings

คำสั่ง การทำงาน
Space + ff ค้นหาไฟล์
Space + e เปิด File Explorer
Space + th เปลี่ยนธีม
Space + ch ดู keybindings ทั้งหมด
